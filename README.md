# CovidApp

### Description

A Flutter mobile app that mimics the covid app tracker. 



### Setup

1. **Clone the Repository:**

    ```bash
    git clone https://github.com/carlosred/CovidApp.git
    cd CovidApp
    ```

2. **Install Dependencies:**

    ```bash
    flutter pub get
    ```

### Running the App

#### Android

```bash
flutter run -d android
```
